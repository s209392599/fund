基金稳定性评估指南

一、分红干扰的解决：复权净值计算

官方计算公式

plaintext
当日复权净值 = 前一日复权净值 × (1 + 日净值增长率)


基金成立首日的前一日复权净值为1.0000元
日净值增长率保留小数点后8位以提高精度

手动计算示例

日期	单位净值	日净值增长率	复权净值
2025-02-24	1.0000	0%	1.0000
2025-02-25	1.0100	1.0000%	1.0100
2025-02-26	1.0150	0.4950%	1.0150

免费获取渠道

天天基金网、蛋卷基金等平台提供"复权净值"数据
中国银河证券基金研究评价网站（www.yhzqjj.com）提供专业版复权净值数据

二、三大核心指标评估基金稳定性

1. 波动率（Volatility）

定义：衡量基金净值波动的剧烈程度，通常用年化标准差表示
计算步骤：
计算每日收益率：当日收益率 = (当日净值 - 前一日净值) / 前一日净值
计算收益率的标准差
年化波动率 = 日收益率标准差 × √252
判断标准：
优秀稳健型基金：年化波动率 < 10%
中等稳健型基金：10% < 年化波动率 < 15%
高波动基金：年化波动率 > 15%

2. 最大回撤（Max Drawdown）

定义：基金净值从最高点到最低点的最大跌幅
计算公式：
plaintext
最大回撤 = (最高点净值 - 最低点净值) / 最高点净值 × 100%


判断标准：
优秀稳健型基金：最大回撤 < 5%
中等稳健型基金：5% < 最大回撤 < 10%
高波动基金：最大回撤 > 10%

3. 夏普比率（Sharpe Ratio）

定义：衡量每承担一单位风险所获得的超额收益
计算公式：
plaintext
夏普比率 = (基金年化收益率 - 无风险利率) / 年化波动率


（无风险利率通常用10年期国债收益率代替，当前约为2.5%）
判断标准：
优秀稳健型基金：夏普比率 > 2
中等稳健型基金：1 < 夏普比率 < 2
低效率基金：夏普比率 < 1

三、实战评估流程

1. 数据准备

收集基金过去1年的复权后累计净值数据
计算每日收益率：当日收益率 = (当日净值 - 前一日净值) / 前一日净值

2. 指标计算

使用Excel的STDEV.P函数计算日收益率的标准差，再乘以√252得到年化波动率
遍历净值数据，找到最高点和最低点，计算最大回撤
计算基金年化收益率，减去无风险利率，再除以年化波动率得到夏普比率

3. 综合评估

如果年化波动率 < 10%，最大回撤 < 5%，夏普比率 > 2，则可以认为是"稳定向上增加"的基金
如果年化波动率 < 15%，最大回撤 < 10%，夏普比率 > 1，则可以认为是"中等稳健"的基金

四、注意事项

分红复权的重要性：如果不进行复权处理，分红会导致净值曲线出现人为的下跌，从而高估波动率和最大回撤
样本周期的选择：1年的周期可能不足以全面评估基金的稳定性，建议至少观察3年以上的表现
市场环境的影响：在熊市中，即使是稳健型基金也可能出现较大回撤，需要结合市场环境进行分析
